    Function fromString

    Deserializes data from a string.

    T fromString(T, Format, siryul.common.DeSiryulize flags = DeSiryulize.none, U) (
      U str
    )
    if (isSiryulizer!Format && isInputRange!U);

    String is assumed to be UTF-8 encoded.

    Parameters

    NameDescription
    T Type of the data to be deserialized
    Format Serialization format (YAML, JSON, AutoDetect)
    str A string containing serialized data in the specified format Supports structs, built-in data types, Nullable, and std.datetime structs.

    Returns

    Data contained in the string

    Example

    struct TestStruct {
	string a;
}
//Compare a struct serialized into two different formats
const aStruct = fromString!(TestStruct, JSON)(`{"a": "b"}`);
const anotherStruct = fromString!(TestStruct, YAML)("---\na: b");
assert(aStruct == anotherStruct);
