    Function toString

    Serializes data to a string.

    auto auto toString(Format, siryul.common.Siryulize flags = Siryulize.none, T) (
      T data
    ) @property
    if (isSiryulizer!Format);

    UTF-8 encoded by default.

    Supports structs, built-in data types, Nullable, and std.datetime structs.

    Parameters

    NameDescription
    Format Serialization format (YAML, JSON, AutoDetect)
    data The data to be serialized

    Returns

    A string in the specified format representing the user's data, UTF-8 encoded

    Example

    //3 as a JSON object
assert(3.toString!JSON == `3`);
//"str" as a JSON object
assert("str".toString!JSON == `"str"`);
